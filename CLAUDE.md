# ELABORA Servicios

Web de servicios y contactos de empresa.

## Stack

- HTML, CSS y JavaScript puro — sin frameworks ni dependencias externas.

## Estructura

- `index.html` — archivo principal y punto de entrada de la aplicación.
- `style.css` — estilos de la web.
- `script.js` — lógica de la aplicación. Contiene los datos de servicios y contactos.
